Рекомендованный способ создания проектов с использованием Мирты.

## Установка

Не требуется.

## Использование

В целевом расположении открыть консоль и выполнить команду

```sh
pnpm create mirta
```

По умолчанию, команда запускает мастер создания проекта в интерактивном режиме и задаёт ряд вопросов. Вы можете пропустить эти вопросы, указав соответствующие опции в качестве аргументов. Отобразить полный список опций:

```sh
pnpm create mirta --help
```
