Конфигурации TypeScript для проектов Мирты.

## Установка

```sh
pnpm add -D @mirta/tsconfig
```

## Использование

Добавьте следующую строчку в ваш `tsconfig.json`:

```json
"extends": "@mirta/tsconfig"
```
